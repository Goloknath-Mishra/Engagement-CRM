# User Manual

## 1. Login

- Open the web app.
- Log in with your username/password.
- After login, the navigation drawer shows modules by area (Sales, Customer Service, Product, Tools, Settings).

## 2. Accounts (Companies)

- Navigate: Customers → Accounts
- Click an account row to open the **Account Detail**:
  - Company hierarchy (parent/subsidiary)
  - Org chart (leadership/managers)
  - Attachments
  - Audit history (admin-only)

### Employees

- From Accounts list, click **Employees**.
- This opens Contacts filtered to that company.

## 3. Contacts (Employees)

- Navigate: Customers → Contacts
- Contacts show:
  - Color identity avatar
  - Company
  - Manager
  - Relationship tag (Decision maker / Influencer / Blocker / Unknown)

Edit a contact to:
- Assign a company (Account)
- Assign a manager
- Set relationship tag
- Upload attachments

## 4. Sales

### Campaigns

- Navigate: Sales → Campaigns
- Open a campaign to:
  - view/edit details
  - view related leads
  - upload attachments
  - link Knowledge Articles
  - view audit history

### Leads

- Navigate: Sales → Leads
- Open a lead to:
  - upload attachments
  - link Knowledge Articles
  - view audit history

### Opportunities

- Navigate: Sales → Opportunities
- Switch between list and kanban.
- Open an opportunity to:
  - manage line items
  - manage contacts on the opportunity
  - upload attachments
  - view audit history

## 5. Customer Service

### Cases

- Navigate: Customer Service → Cases
- Open a case to:
  - see progress indicator and SLA metrics
  - link Knowledge Articles
  - insert templates into description while editing
  - upload attachments
  - view audit history

### Incidents (War Room)

- Navigate: Customer Service → Incidents (War Room)
- Open an incident to:
  - post messages
  - upload attachments
  - view audit history

## 6. Product Catalog

- Navigate: Product → Catalog
- Create/edit products.
- Upload attachments and view audit history in the edit dialog.

## 7. Gamification

- Navigate: Overview → Gamification
- Switch module tabs: Sales / Service / Marketing
- Join challenges and track progress and leaderboards.
- View badge catalog and awards; share badges.

## 8. Knowledge Articles

- Navigate: Tools → Knowledge Articles
- Create and publish articles.
- Link articles from Case/Lead/Campaign record pages.

## 9. Templates

- Navigate: Tools → Templates
- Create templates by type.
- Use templates from Case edit screen (insert/copy).

## 10. Reports

- Navigate: Tools → Reports
- Create report definitions:
  - entity type
  - columns
  - filters
- Preview a report.
- Export to CSV/Excel/Word/PDF.
- Share report via email.
